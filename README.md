# はるのさと サイト — ファイル構成ガイド

## 分割の目的
元の `harunosato.html`(約762KB・1行53万文字のbase64画像を含む)は、AIでの編集時に
処理エラーが頻発するサイズでした。画像・CSS・JSを外部ファイル化し、
編集対象のHTMLを **約62KB / 890行** まで縮小しています。

## ファイル構成
```
harunosato/
├── index.html        … ページ本体(HTML構造のみ・約62KB)
├── css/
│   └── style.css     … 全スタイル(旧<style>ブロック3箇所を統合)
├── js/
│   └── script.js     … 全スクリプト(モーダル開閉・ナビ等)
└── images/
    ├── hero-bg.jpg        … トップのヒーロー背景(CSSから参照)
    ├── service-room.jpg   … サービス紹介「住まい」カード
    ├── service-day.jpg    … サービス紹介「デイサービス」カード
    ├── service-kaigo.jpg  … サービス紹介「訪問介護」カード
    └── staff-suzuki.jpg   … 鈴木良明氏 写真(2箇所で共用・重複を統合済み)
```

## 編集時の注意
- **文章・構造の修正** → `index.html` のみ編集すればOK
- **デザイン変更** → `css/style.css`(色は `:root` のCSS変数にまとまっています)
- **動作の変更** → `js/script.js`
- **画像の差し替え** → `images/` 内のファイルを同名で置き換えるだけ
  (HTMLを触る必要なし)
- ヒーロー背景はCSS内で `../images/hero-bg.jpg` と相対参照しています
  (CSSファイル基準のパスのため `../` が必要)

## 公開時
4ファイル+imagesフォルダを**同じ階層構成のまま**サーバーへアップロードしてください。
ローカル確認はフォルダごと保持した状態で `index.html` をブラウザで開けば動作します。
